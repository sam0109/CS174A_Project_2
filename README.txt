I implemented the whole project plus both extra credit portions.
The keys to navigate are as follows:
w: forward
s: back
a: left
b: right
e: move up
q: move down
left arrow: turn left
right arrow: turn right
up arrow: turn up
down arrow: turn down
r: reset view
t: attach view to the 4th planet

My sphere function can generate smooth normals if the smooth variable is set to true, but I chose to 
generate the smooth normals in the shader only if the sphere being used needed them.
